# PrivTF: Efficient and Privacy-Preserving Transformer Inference with Optimized Protocols for Secure NLP

This repository contains a reference implementation of **PrivTF**, based on the TNSE'25 work on privacy-preserving Transformer inference with optimized secure protocols.

[Paper](https://ieeexplore.ieee.org/document/11150591)

## Setup and Usage

The implementations are based on the EzPC framework. Please refer to the [EzPC repository](https://github.com/mpc-msri/EzPC) for setup instructions and documentation.

## Usage Statement for Research Purpose

- This repository is primarily intended for academic research and experimental reproduction.
- Some components are experimental and may not be production-ready.
- For reproducibility, please pin dependencies and record your runtime configuration.
